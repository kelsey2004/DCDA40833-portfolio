Image Carousel

With the assistance of AI, I was able to create two carousels in my Lab 2 for each of the sections on this page. Now when you visit that page you can easily navigate through the images with the arrow buttons. Additionally,there is now the option to view the carousel in fullscreen mode. In order to achieve this, I used AI to generate the necessary JavaScript and CSS code for the carousels. I then integrated this code into my existing HTML and CSS files, using the prompt down below:

1.Don't change anything else. In Lab02.html, add two image carousels. One in section Tool 1: Adobe Podcasts and the other in section Tool 2: Pika. It should have arrows to click back and forth. The figure captions associated with the figures should also change when the image changes.

After the first prompt, AI created “Carousel-slide” to wrap each figure and figure caption, having them be read as a pair so when I move through the images, the caption would also switch. It created the buttons that would be visible on the page. When clicked, they would either take me to the previous, next, expand, or switch the dots for navigation. The option for an expand button came after modifying when I realized that the carousel would shrink to adjust to the changing width of the webpage. If one were to view it on a mobile device, it would be harder to see. Because of this I added a expand button that allows for a full screen view. I did this using the second prompt below:

2. I like what you did, don't change anything else. Now add an expand button that takes the carousel into a fullscreen view.

This create the image carousel that is easy to navigate even in a mobile view.

Bar Chart Visualization

I used Google Colab and AI to help me visualize my dataset into a bar chart. I've worked with python, but my skills are a little rusty. I was interested in the relationship between phone usage and age so I tasked AI with creating a simple bar. It grouped the Daily_Phone_Hours by the Age because it needed to look at the phone hours within each age in order to calculate the sum for each. Then it used the matplotlib package to turn this data into a bar chart. I did this using the prompt below:

1. Read the CSV file Smartphone_Usage_Productivity_Dataset_50000.csv and then create a bar chart that visualizes the total Daily_Phone_Hours on the Y axis and Age on the X axis.

The first output was nearly perfect but not every age was visible on the X-axis. The number visible were every 10 (e.g. 20,30,40) so I modified the prompt to show every age.

2. I like what you did. Without changing anything else, make every single age value on the X-axis visible.

The final outputted a bar chart that showed the total phone hours according to the age group.

My Vibe Coding Guidelines

In the future, when it comes to working with AI, as long as it is not penalized in my academic work or professional work, then I'd like to continue experimenting with AI, especially if it helps me enhance my work. As long as I am fully transparent to others and in my work. It is dangerous to assume AI is always right so I will never let AI make decisions for me, think on my behalf, nor generate large-scale projects. I should not use AI if I cannot at least understand the logic. I should at least ask AI to explain what I am implementing.
